 🗣️ Lingua AI

**Impara a parlare male una lingua** — Traduzione istantanea, pronuncia realistica e input vocale per imparare l'inglese, spagnolo, francese e tedesco.

## ✨ Funzionalità

- **Traduzione** da italiano a 4 lingue (inglese, spagnolo, francese, tedesco) tramite API gratuita Lingva.
- **Pronuncia automatica** della frase tradotta (velocità regolabile).
- **Input vocale** in italiano (dettatura della frase da tradurre).
- **Ascolto della traduzione** con un semplice clic.
- **Design moderno** e reattivo (funziona su mobile e desktop).
- **PWA pronta** (manifest, service worker, icone) – installabile come app nativa.

> ⚠️ **Nota sulla pratica vocale**: il riconoscimento vocale per ripetere la frase tradotta è disponibile **solo in inglese** a causa dei limiti della Web Speech API nei browser. Per le altre lingue, puoi comunque ascoltare la pronuncia corretta.

## 🚀 Come avviare l'app in locale

### Prerequisiti
- Node.js (versione 18 o superiore)
- npm (o yarn)

### Passi

```bash
# Clona il repository (o scarica i file)
git clone https://github.com/tuo-username/lingua-ai.git
cd lingua-ai

# Installa le dipendenze
npm install

# Avvia il server di sviluppo
npm run dev
Apri http://localhost:5173 nel browser.

Build per la produzione
bash
npm run build
La cartella dist conterrà i file statici da caricare su qualsiasi hosting (Netlify, Vercel, Cloudflare Pages, ecc.).

🛠️ Tecnologie utilizzate
React 18 + Vite

Lucide React per le icone

Web Speech API (riconoscimento vocale e sintesi vocale)

Lingva API (traduzione gratuita, senza chiave)

CSS-in-JS (stili inline, nessuna dipendenza esterna)
