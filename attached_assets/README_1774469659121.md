# 🎣 Diario del Pescatore — v3.6

App web PWA per pescatori del Mar Tirreno. Registra uscite, catture, spot e attrezzatura — tutto salvato in locale nel browser con sync cloud automatico. Installabile su Android e iOS come app nativa.

> *"Ad Maiora Semper"* — by Limax

---

## 🚀 Avvio rapido

Richiede **Node.js 18+**.

```bash
cd diario-pescatore
npm install
npm run dev
Apri il browser su http://localhost:5173

📱 Deploy PWA
bash
npm run build
npx wrangler pages deploy dist --project-name=diario-pescatore --commit-dirty=true
Su Android Chrome → menu ⋮ → "Aggiungi a schermata Home"
Su iOS Safari → Condividi → "Aggiungi a schermata Home"

🤖 Assistente AI (Chat)
L'AI può rispondere a domande di pesca e salvare dati nel diario tramite linguaggio naturale.

Provider	Note
DeepSeek	Unico provider usato (chiave inclusa nell'app)
Esempi:

"Oggi ho pescato una spigola da 2kg a Terracina" → salva cattura

"Registra uscita surfcasting a Sabaudia, mare calmo" → salva uscita

"Salva spot: Secca del Faro, fondale roccioso, 15m" → salva spot

Voce: 🎙 microfono, trascrizione in italiano via Web Speech API (Chrome/Android).

📷 Scanner specie (manuale)
Disponibile tramite il pulsante 🔵 Scan (mobile) e nella Home.
Scatta o carica una foto, quindi compila i campi specie, peso, lunghezza, spot, note.
Salva la cattura con foto allegata. Nessuna chiave API richiesta.

🎣 Previsioni Pesca
Card interattiva in Home + pagina completa. Ottimizzata per pesca da terra sul Canale Fiume Portatore e foce di Porto Badino.

Algoritmo — modello moltiplicativo con vincoli
Livello	Descrizione
Blocchi duri	Temporale → 4pt · Acqua gelida → 6pt · Mare agitato → 8pt
Punteggio base	Somma pesata dei 6 fattori (pesi non uniformi)
Moltiplicatore	Luna × Maree si amplificano a vicenda (fino a +15 pt)
Bonus combinati	Alba + marea · Notte + luna nuova · Pioggerella + foce
Penalità soft	Acqua fredda · Vento forte · Marea stagnante
Pesi dei fattori
Fattore	Peso	Fonte
Maree (velocità variazione)	30 pt	Modello armonico M2+K1+M4
Luna (fase + distanza)	25 pt	Calcolo matematico (no API)
Temperatura acqua	20 pt	Open-Meteo Marine SST
Onde / mare	10 pt	Open-Meteo Marine
Vento	9 pt	Open-Meteo forecast + direzione rispetto alla foce
Meteo cielo	6 pt	Open-Meteo WMO code
Card Home — 6 tab interattivi
📊 Dati — 7 fattori + vento rispetto alla foce + periodi solunar + grafico orario

🎣 Tecnica — tecniche da terra ordinate per condizioni con specie target

🪱 Esche — 5 esche consigliate per trasparenza, SST, stagione, ora

🐟 Specie — 11 specie di Porto Badino con probabilità in tempo reale e zona

🌊 Campo — input condizioni reali (livello canale, torbidità, corrente foce, note)

🤖 AI — consiglio contestuale DeepSeek (deepseek-chat) basato su tutti i dati

Tab 🤖 AI
DeepSeek riceve vento + direzione foce, onde, temperatura, maree, luna, solunar, trasparenza e i dati del tab Campo. Risponde come pescatore esperto di Porto Badino con consiglio su orario, esca, posizione e specie.

Tab 🌊 Campo — correzione campo reale
Campo	Effetto punteggio
Livello canale (Basso/Normale/Alto/Piena)	da −15 a +5 pt
Torbidità (Chiara/Normale/Torbida/Fangosa)	da −14 a +8 pt
Corrente foce (Ferma→Forte)	da −6 a +4 pt
Pesci visibili in superficie	+12 pt + cambia esche
Note libere	testo salvato, no effetto numerico
Specie target Porto Badino
Su tutto il Portatore: Granchio blu · Spigola · Serra · Cefalo · Muggine · Mormora · Ombrina · Sogliola · Anguilla · Sparaglione

Alla foce (+ prima fascia): Leccia stella · Orata

Tecniche da terra (nessuna tecnica da barca)
Tecnica	Specie target
Surfcasting foce	Spigola, Serra, Orata, Leccia stella, Ombrina
Feeder / fondo canale	Cefalo, Anguilla, Sogliola, Mormora, Sparaglione
Spinning da riva	Spigola, Serra, Leccia stella
Bolognese / colpo	Cefalo, Muggine, Sparaglione, Mormora
Fondo notturno	Anguilla, Spigola, Sogliola
Pesca al granchio blu	Granchio blu (tutto il Portatore + foce)
Stazioni disponibili
Civitavecchia · Fiumicino · Anzio · Nettuno · Borgo Grappa · Sabaudia · S.F. Circeo · Foce Sisto · Terracina · Porto Badino (default) · Sperlonga · Gaeta · Formia

Nota: La località scelta in Meteo, Maree o Previsioni Pesca viene salvata in localStorage e condivisa tra tutte le sezioni.

💾 Salvataggio dati
Dove	Come	Quando
localStorage	Copia locale nel browser	Istantaneo ad ogni modifica
JSONBin.io	Sync cloud	Automatico ogni 45s e ad ogni modifica
Protezione race condition: il push su cloud è bloccato finché il pull iniziale non è completato. In caso di conflitto vince chi ha più record; in parità vince il timestamp più recente.

La sincronizzazione è visibile nella sidebar (badge Sync) con notifiche toast:

✅ Verde — dati sincronizzati

⚠️ Giallo — errore sync, retry automatico

❌ Rosso — sync non disponibile, dati in locale

🔵 Blu — aggiornamento da altro dispositivo
